How it works (5 steps)

  1. Set bias with SP‑EC Pro and/or SP‑MACD+.
  2. Verify participation via PVA Overlay + OBV/CVD.
  3. Map structure with SDZ and Levels.
  4. Time entries with RSI Triad / SRSI+ and LTF rhythm.
  5. Scale & alert using SP‑Screener (on‑chart or global).

Built‑in plays

  • Trend Buy‑the‑Dip — Bias Up + PVA Bull/Climax → reaction at demand/dVWAP → RSI Triad/SRSI+ timing.
  • Reversal Flip — Absorption + OBV divergence at supply → EC slope flatten → Pilot reclaim.
  • Breakout Continuation — Climax through wVWAP/supply → first ATR dip → NanoFlow “+”.

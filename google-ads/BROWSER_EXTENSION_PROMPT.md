# 8. TROUBLESHOOTING

### If ads get disapproved:
```
My Google Ad was disapproved for [PASTE REASON HERE].

Here's my ad copy:
Headlines: [PASTE YOUR HEADLINES]
Descriptions: [PASTE YOUR DESCRIPTIONS]
URL: [PASTE YOUR URL]

The product is Signal Pilot - TradingView indicators for technical analysis. It's educational trading tools, NOT financial advice or investment services.

How can I modify my ad copy to comply with Google Ads policies while keeping the core message?
```

### If Quality Score is low:
```
My keyword "[PASTE KEYWORD]" has a Quality Score of [NUMBER]/10.

My ad headlines include: [PASTE 2-3 HEADLINES]
My landing page is: signalpilot.io

Google says the issue is: [Expected CTR / Ad Relevance / Landing Page Experience]

What specific changes can I make to improve this Quality Score?
```

### If CPC is too high:
```
My average CPC for "[PASTE KEYWORD]" is $[AMOUNT] which is too high for my budget.

Current match type: [exact/phrase/broad]
Current bid strategy: [Maximize Clicks / Manual CPC / etc.]

What can I do to lower my CPC while maintaining relevance? Should I:
- Change match types?
- Use different keywords?
- Adjust bids?
- Try different ad copy?
```

### If conversions aren't tracking:
```

My conversion action is set to:
```

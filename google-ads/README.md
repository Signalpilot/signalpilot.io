## Key Metrics to Track

1. **Trial Signups** - Primary conversion
2. **Cost Per Trial** - Target: $10-25
3. **Trial-to-Paid Rate** - Monitor monthly
4. **ROAS** - After tracking purchases

---

## Getting Started

1. Create Google Ads account
2. Set up conversion tracking (GTM recommended)
3. Create Campaign 1 (English) first
4. Add all ad extensions
5. Launch with $30-50/day budget
6. Monitor for 2 weeks, then optimize
7. Launch Campaign 2 (Localized) after English is stable

## Notes

- All ad copy complies with Google Ads policies

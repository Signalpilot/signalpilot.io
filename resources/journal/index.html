Trading Journal Template

A simple source of truth for each trade. This CSV includes columns to log confidence multipliers, risk, and outcomes.

CSV Template

Columns included:

  • Date, Symbol, Direction, Entry, Stop, Target, Size
  • Session, Setup, ConfidenceScore, Reason, Tags
  • Exit, PnL, RMultiple, Outcome, Notes, ScreenshotURL

The CSV can be opened in Excel/Sheets or imported to journaling tools. ConfidenceScore reflects module confluence from SignalPilot.
